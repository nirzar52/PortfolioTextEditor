# PortfolioTextEditor
This text editor app is built using stack data structure.
A terminal-based text editor built to demonstrate my understanding of Stacks in data structure & algoriths course.

## Features
- **Custom Stack Implementation:** Implements generic `IStack<T>` and `IContainer<T>` interfaces using dynamic linear array.
- **Undo/Redo:** Uses push and pop stack methods to track and navigate history.

## Data Structure Logic
When the stack reaches capacity, it executes `DoubleCapacity()` to manage memory allocation.
